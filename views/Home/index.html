Things i have learned

This is pretty much a basic website, where you can login and register. I did most of the error management, if missed some, don't hesitate to make a pull request so you can as well upgrade the example for future beginners!

Beego API Framework

I learned how to manage routes, make POST and GET request, catch a JSON body from a POST request and make the Beego Framework work with the MVC project architecture

Beego ORM

Beego ORM is pretty easy to communicate with databases, i learned how to get informations from rows, insert a new column, create tables, remove tables, delete database and many mores.

FrontEnd Communication

This example helped me to enhance my skills in Ajax, Javascript and JQuery, the languages i used to make the frontend and backend communicate.

Docker Local Database

I found a new way to make a local databse threw Docker Compose.
